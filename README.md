# OMナビ 静的ツール集

GitHub Pagesで公開する、サーバー不要のツール集。

| ページ | 内容 |
|---|---|
| `index.html` | 工務店コード検索（あいまい検索・コードのワンクリックコピー・★お気に入り/よく使う） |
| `call.html` | 通話文字起こし・自動転記ツール（ステレオ通話対応） |

## データについて

- 工務店名簿は `data.js` に **AES-256-GCM で暗号化** して格納しています。ページを開くとパスワード入力を求められ、正しい場合のみブラウザ内で復号されます（平文データはリポジトリに含まれません）。
- ★お気に入り・利用履歴は各自のブラウザ（localStorage）に保存されます。端末・ブラウザごとに別々です。

## データの更新方法

工務店名簿を更新するときは、ローカルで:

```bash
# 1. 最新の 工務店情報全項目.csv から平文JSONを作る（OMナビ本体リポジトリのスクリプトを利用）
# 2. 暗号化して data.js を再生成
node tools/encrypt.mjs tools/_plain.json <パスワード>
# 3. data.js をコミット・push（_plain.json はコミット禁止）
```
